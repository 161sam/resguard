# Resguard v0.3.0 Release Notes

## Highlights

- Snap-aware desktop discovery hardening for Ubuntu desktop systems.
- Practical `resguard suggest --apply` flow for common Firefox/VS Code snap workloads.
- Safer and clearer desktop wrapper behavior (`DBusActivatable=false` in wrappers, conservative fallback when no unique desktop id exists).
- Operator UX polish for `doctor`, `status`, and `metrics` output on real installed systems.

## Packaging and install

- Core package: `resguard_<version>_amd64.deb`
- Optional daemon package: `resguard_<version>_amd64_daemon.deb`
- APT repository publishing remains tag-driven via GitHub Actions.

## Notes

- `resguardd` remains optional and is not installed by default in the core package.
- Additional GNOME/KDE desktop-matrix field validation remains tracked as v0.3.x follow-up.
