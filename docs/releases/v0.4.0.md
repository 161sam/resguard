# Resguard v0.4.0 Release Notes

## Highlights

- First policy/runtime autopilot flow for adaptive pressure handling.
- `resguardd` now executes through services with structured tick decisions and safer default behavior.
- Feature-gated TUI now provides practical operator visibility:
  - PSI pressure overview
  - class slice limits and current usage where available
  - recent daemon/autopilot actions from ledger

## Packaging and install

- Core package: `resguard_<version>_amd64.deb`
- Optional daemon package: `resguard_<version>_amd64_daemon.deb`
- APT repository publish remains tag-driven via GitHub Actions.

## Notes

- `resguardd` remains optional and is not installed by default in the core package.
- TUI remains feature-gated and is currently intended for source builds (`--features tui`).
